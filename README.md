# 💻 Desafio de Programação — POO com Python

Projeto desenvolvido com base no desafio proposto pela comunidade SysNight, com foco na prática de Programação Orientada a Objetos (POO) em Python.

## 📚 Sobre o projeto

O objetivo foi criar uma classe `Personagem` contendo atributos e métodos que simulam uma batalha simples entre personagens.

Foram trabalhados conceitos como criação de classes, objetos, métodos de instância e interação entre objetos, além da utilização da biblioteca Rich para uma exibição mais visual no terminal.

## ▶️ Estrutura do desafio

* Criar personagens com nome, vida e ataque
* Exibir status atual
* Realizar ataques entre personagens
* Receber dano
* Verificar quando um personagem foi derrotado

## 🎯 Objetivo acadêmico

Treinar os fundamentos da Programação Orientada a Objetos em Python de forma prática e aplicada, fortalecendo a lógica de programação e a organização de código.
